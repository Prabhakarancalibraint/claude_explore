# Debug Template

**Role**: You are a senior [language] engineer with expertise in debugging and root cause analysis.
**Context**: Working on [project name] using [technology stack - e.g., React, Node.js, PostgreSQL].
**Task**: Debug the following issue:
[Describe the bug/error - include error message, stack trace, or unexpected behavior]

File: [file path]
Function: [function name]
Line: [line number if known]
**Constraints**:
- Do not change test files
- Maintain existing API contracts
- Fix root cause, not symptoms
- Do not add new dependencies
**Output Format**:
1. Explain the root cause in 1-2 sentences
2. Provide the fix with exact file paths and line numbers
3. Include verification steps to confirm the fix works
