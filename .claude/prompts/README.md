# Personal Prompt Template Library

This directory contains reusable prompt templates using the 5-Part Formula.

## Quick Reference

| Template | Use For |
|----------|---------|
| debug.md | Bug fixing, error investigation |
| create-component.md | React/Vue components |
| create-api.md | REST API endpoints |
| refactor.md | Code refactoring tasks |
| review.md | Code review requests |

## Adding New Templates

1. Copy the template format below
2. Save as `[category].md`
3. Customize the placeholders for your workflow

## Template Format

```markdown
**Role**: [Who Claude should be]
**Context**: [Project/codebase background]
**Task**: [Exactly what to do]
**Constraints**: [What NOT to do]
**Output Format**: [How the result should look]
```
