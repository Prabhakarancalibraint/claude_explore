# Hello World Skill

A simple demonstration skill that shows the basic structure of a skill.

## When to Use

- Testing skill invocation
- Learning skill structure
- Demonstrating skill capabilities

## What This Skill Does

This skill will respond with a friendly greeting and demonstrate how skills work.

## Example Output

```
Hello! This is a demonstration skill.

Here's what a skill can do:
1. Provide structured responses
2. Follow specific formats
3. Encapsulate domain expertise
4. Automate workflows

Skill invocation successful!
```

## Key Points

1. Skills use YAML frontmatter with name and description
2. The description determines when the skill triggers
3. Instructions in the body guide the skill's behavior
4. Skills can include bundled resources (scripts, references, assets)

---

## How to Extend This Skill

To create a more useful skill:
1. Define a clear purpose
2. Write specific trigger conditions
3. Add detailed instructions
4. Include examples
5. Test with realistic prompts
